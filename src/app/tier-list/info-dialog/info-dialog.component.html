How to use Tier Lister

Editing the Board

Use the edit button to change the name and author of the board.

Adding Items

Use the add button to add an item. You can enter a URL to use an image, or you can just name the item. Ensure you use a direct link to the image or it will not work.

Deleting Items

Drag any item down below the None tier to delete it.

Downloading Tier Lists

Use the get_app button to download a tierlist to a file. You can upload these to access them. This lets you save boards for later, as well as creating templates.

Adding Items

Use the publish button to upload a tierlist. Drag the file into the upload area, then select it from the menu to view it.
